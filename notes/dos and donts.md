### 少有人走的路

只要选择困难模式的路，绝对是少有人走的路。

| do（困难的） | don't（容易的） |
| ------------ | --------------- |
| 专心工作     | 摸鱼            |
| 生产效率     | 说大话          |
| 整洁         | 混乱            |
| 努力         | 享受            |
| 骑行         | 开车            |
| 准备         | 随遇而安        |
| 仔细检查     | 粗心大意        |
| 写作         | 聊天            |
| 工作         | 游戏            |
| 耐心，慢     | 急于求成        |
